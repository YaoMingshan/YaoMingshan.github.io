Cutlass Terminology and Abstraction

2025-08-10 · 5 min · 990 words

Terminology

  • N - Column Major Matrix
  • T - Row Major matrix
  • {N,T} x {N,T} - All combinations, i.e., NN, NT, TN, TT
  • lda: leading dimension of A
  • ldb: leading dimension of B
Name T N
A [M, K]
lda:K		 [K, M]
lda:M
B [K, N]
ldb:N		 [N, K]
ldb:K
C [M,N]
ldc:N		 [N,M]
ldc:M

以下内容是Cutlass中的术语和抽象，不包含Cute

Layout

functor mapping logical coordinates of a tensor to linear offset (as LongIndex); owns stride vectors.它只包含stride信息，主要是将坐标信息转换成一维的offset，和cute是不一样的，cute包含shape+stride信息。

  • Gemm特化的layout： RowMajor and ColumnMajor

以RowMajor为例

  • 它是2维的（Rank）
  • stride为1维
  • Index为多维坐标维度类型
  • LongIndex是将多维转换为线性偏移时的类型
  • TensorCoord用了MatrixCoord，它是一个两维的坐标
  • stirde用了一个1维的Coord去表达
  • Coord就是存储了N维的坐标

Element

表达数据类型，比如ElementA: float

Extent

每个维度的大小

Fragment

一片寄存器tile

Basic gemm

代码cutlass/examples/00_basic_gemm/basic_gemm.cu

  • 构造Gemm gemm_operator，传入的参数是编译期的模版参数
    • A、B、C、Acc的数据类型
    • A、B、C Layout
    • ArchTag架构信息
    • block shape
      • 架构和数据类型确定时，都用了固定的切分，常见<128, 128, K> <128, 256, K>
    • warp shape
      • 架构和数据类型确定时，都用了固定的切分，常见<32, 64, K> <64, 64, K>
    • instruction shape
    • epilogue op
    • pipeline stage数量
    • memory对齐信息
    • 根据以上信息选择GemmKernel，cutlass/include/cutlass/gemm/kernel/default_gemm.h
      • Gemm, cutlass/include/cutlass/gemm/kernel/gemm.h
        • ThreadblockMma，和架构相关
          • mmapipelined
            • mma core
              • 计算warp数量，block thread数量
            • IteratorA: global to shared数据搬运
            • IteratorB
            • SmemIteratorA：shared to 寄存器数据搬运
            • SmemIteratorB
            • operator()中执行2d的load和计算
              • prologue: 从global load数据到shared
              • gmem_wait
              • gemm_iter
                • 循环从global load数据到shared
                • 循环从shared load数据到寄存器
                • warp_mma
        • epilogue
        • threadblockswizzle
        • 在gemm的operator()是功能实现入口、包含了计算block起始位置、数据搬运、计算、后处理
  • 构造Arguments arg，它类型在Gemm中定义了，参数都是运行期kernel参数
    • MNK

    • alpha/beta epilogue参数
  • launch kernerl，gemm_operator(arg)
    • 调用initialize准备kernel参数
